Veda Reader UI exploration

Three layout directions for a lighter reader shell.

These HTML mockups show ways to reduce top-toolbar weight on desktop and mobile while preserving document navigation, search, display controls, and a large reading canvas. The document content is only illustrative.

Recommended direction: Option B on desktop, Option C on mobile

Desktop gets the most immersive reading experience with auto-hiding chrome. Mobile gets the clearer split between navigation and actions, using a docked bottom sheet instead of a persistent heavy toolbar.
